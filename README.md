# List-webapp

A modern and responsive list management web application built with HTML and CSS.

## Features

- Clean and modern user interface
- Responsive design for all devices
- Organized project structure
- Lightweight and easy to customize
- Fast loading static website

## Project Structure

```
list-webapp/
├── index.html
└── assets/
    ├── css/
    │   └── style.css
    └── img/
        └── logo.png
```

## Technologies Used

- HTML5
- CSS3

## Getting Started

Clone the repository:

```bash
git clone https://github.com/sara-ghaffarain/list-webapp.git
```

Open `index.html` in your browser.
